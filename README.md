# Android-PhotoAlbum  사진 앨범 어플

⚙️ **Tool : Android Studio, Java, Gradle**

<br>

## 📑 요구사항
**1. 30장의 사진이 들어있는 앨범 어플을 만든다**

**2. 이전, 다음 버튼을 만들어 사진을 넘길 수 있다.**

**3. 사진의 넘버를 입력하면 해당 사진으로 이동할 수 있다.**

<br>
